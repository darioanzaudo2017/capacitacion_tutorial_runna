Cómo usar esta guía

Esta guía contiene ejercicios prácticos para aprender el sistema RUNNA trabajando con casos simulados. Todos los ejercicios se realizan con tu usuario administrador y datos ficticios preparados para la capacitación.

Cada ejercicio tiene tres partes:

  • Situación: el contexto del caso en lenguaje cotidiano, como lo recibirías en la realidad.
  • Pasos en el sistema: instrucciones detalladas de qué hacer y dónde.
  • Checklist de verificación: para confirmar que completaste cada paso correctamente. Tu progreso se guarda automáticamente en este navegador.
